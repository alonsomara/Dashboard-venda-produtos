# Dashboard-venda-produtos
- Projeto criado no módulo 23 do curso Análise de Dados da EBAC com o intuito de criar uma Dash interativa desenvolvida em Python usando a biblioteca Plotly e Dash.
- Desenvolvimento de gráficos após a criação de Scripts e IDE, coleta, tratamento e preparação de dados, após a aplicação de conceitos básicos de estatística, correlação de dados, modelos de regressão e classificação.
- Criação de Dashboard interativa a partir dos gráficos desenvolvidos no módulo.
  # Funcionalidades
- Filtro por marca
- Filtro por temporada
- Filtro por faixa de preço
- Distribuição de preços
- Relação preço x quantidade vendida
- Mapa de densidade 2D
- Regressão linear
- Correlação entre variáveis
